# 🤖 Intelligent Resume Screening System

## 🚀 Overview
This project automates resume screening by extracting candidate information, scoring applicants based on job requirements, and ranking them efficiently.

## 🔥 Features
- 📄 Parse PDF/DOCX resumes
- 🧠 Extract skills, experience, education
- 📊 Score candidates based on job description
- 🏆 Rank and shortlist top candidates
- 📁 Export results to CSV

## 🛠 Tech Stack
- Python
- PyPDF2
- Pandas
- Regex (NLP basics)

## 📂 How to Run

1. Clone repo

git clone <your-repo-link>
cd resume-screening


3. Install dependencies

pip install -r requirements.txt


3. Add resumes to `/resumes`

4. Run

python main.py


## 📊 Output
- Ranked candidates in terminal
- CSV file in `/output/results.csv`
