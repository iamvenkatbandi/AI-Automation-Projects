# 🚀 AI Email Auto-Responder

An AI-powered email automation system that reads, classifies, and replies to emails automatically using LLMs and Gmail API.

---

## ✨ Features

* 📥 Reads unread emails from Gmail
* 🧠 Classifies emails (support, sales, spam, job, important, ignore)
* ✍️ Generates AI-based replies
* 📤 Sends automated responses
* 🚫 Smart filtering (ignores spam/newsletters)
* ✅ Marks emails as read
* 💾 Prevents duplicate replies
* 📊 Logging system for tracking

---

## 🛠 Tech Stack

* Python
* Gmail API
* OpenRouter (LLM API)
* Requests
* Logging

---

## ⚙️ Setup Instructions

### 1. Clone Repo

```bash
git clone https://github.com/your-username/ai-email-responder.git
cd ai-email-responder
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Add Environment Variables

Create `.env` file:

```env
OPENROUTER_API_KEY=your_api_key_here
```

### 4. Add Google Credentials

* Download `credentials.json` from Google Cloud
* Place in root folder

### 5. Run Project

```bash
python main.py
```

---

## 🔁 Run Automation

```bash
python run.py
```
